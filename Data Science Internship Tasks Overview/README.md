# Data Science Internship Tasks Overview

**Author:** Ayesha Sarwar

Welcome to my internship portfolio repository. This directory contains five distinct machine learning and data science projects (tasks) that I have completed during my internship. Each task focuses on a different aspect of data analysis, exploratory data analysis (EDA), and predictive modeling.

## Task Summaries

### 1. [Task 1: Iris Dataset (`Task1_Iris_Dataset`)](https://github.com/ayeshachaudhary454-lang/DevelopersHub-Corporation-Data-Science-Analytics-Internship-Tasks/tree/main/Task1_Iris_Dataset)
This task involves exploring and analyzing the classic Iris dataset. It includes data loading, exploratory data analysis, data visualization using libraries like Pandas, Matplotlib, and Seaborn, and understanding the different features that distinguish the three species of Iris flowers.

### 2. [Task 2: Credit Risk (`Task2_Credit_Risk`)](https://github.com/ayeshachaudhary454-lang/DevelopersHub-Corporation-Data-Science-Analytics-Internship-Tasks/tree/main/Task2_Credit_Risk)
This project focuses on evaluating credit risk. The goal is to build a predictive model that can assess the likelihood of a borrower defaulting on a loan based on various financial and demographic features. It involves data preprocessing, handling class imbalances, and evaluating classification metrics.

### 3. [Task 3: Churn Prediction (`Task3_Churn_Prediction`)](https://github.com/ayeshachaudhary454-lang/DevelopersHub-Corporation-Data-Science-Analytics-Internship-Tasks/tree/main/Task3_Churn_Prediction)
In this task, the objective is to predict customer churn (i.e., whether a customer will stop doing business with a company). This involves analyzing customer behavior data, identifying key indicators of churn, and building a machine learning model to proactively identify at-risk customers.

### 4. [Task 4: Insurance Prediction (`Task4_Insurance_Prediction`)](https://github.com/ayeshachaudhary454-lang/DevelopersHub-Corporation-Data-Science-Analytics-Internship-Tasks/tree/main/Task4_Insurance_Prediction)
This task is centered around the insurance domain. It involves analyzing customer profiles, demographics, and health attributes to build predictive models that estimate insurance premiums or the likelihood of insurance claims.

### 5. [Task 5: Loan Acceptance (`Task5_Loan_Acceptance`)](https://github.com/ayeshachaudhary454-lang/DevelopersHub-Corporation-Data-Science-Analytics-Internship-Tasks/tree/main/Task5_Loan_Acceptance)
The final task revolves around predicting loan acceptance. The objective is to build a model that can automate or assist the loan approval process by analyzing applicant data (such as income, credit history, loan amount) to determine if a loan application should be approved or rejected.

## Technologies Used
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn
