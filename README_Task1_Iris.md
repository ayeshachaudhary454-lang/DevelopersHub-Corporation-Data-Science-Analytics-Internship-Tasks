# Task 1: Iris Dataset Exploration and Visualization

## Objective
Perform Exploratory Data Analysis (EDA) on the Iris dataset to understand the characteristics of different iris flower species.

## Dataset
- Dataset: Iris Dataset
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
  - Species

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Steps Performed
1. Data Loading
2. Data Cleaning
3. Statistical Summary
4. Data Visualization
5. Correlation Analysis
6. Conclusion

## Results
- Found relationships between flower measurements.
- Visualized species distributions.
- Identified important features for classification.
