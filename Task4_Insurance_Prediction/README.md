# Task 4: Insurance Cost Prediction

## Objective
Predict insurance charges using customer demographic and health information.

## Dataset Features
- Age
- Sex
- BMI
- Children
- Smoker
- Region
- Charges

## Algorithms
- Linear Regression
- Random Forest Regressor

## Evaluation
- R² Score
- MAE
- RMSE

## Conclusion
The model accurately estimates insurance costs and highlights important influencing factors.
