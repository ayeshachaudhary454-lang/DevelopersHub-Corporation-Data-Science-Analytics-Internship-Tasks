# Task 5: Loan Acceptance Prediction

## Objective
Predict whether a customer will accept a personal loan offer.

## Dataset Features
- Age
- Income
- Experience
- Education
- Family Size
- Credit Card Usage

## Machine Learning Process
1. Data Cleaning
2. EDA
3. Feature Selection
4. Model Training
5. Prediction
6. Evaluation

## Models Used
- Logistic Regression
- Random Forest

## Results
The model helps banks identify customers who are more likely to accept loan offers.
